# Claude Code Skills

Claude Code 自定义技能集合。

## Skills

### deep-research
深度调研方法论（8步法+交叉验证）：将模糊主题转化为高质量调研报告。

**触发词**：
- 深度调研、深度研究、深入分析
- 帮我调研、调研一下、研究一下
- 对比分析、概念对比、技术对比
- 写调研报告、出调研报告
- 产品评测、测评、XXX 怎么样
- 竞品分析、竞争对手分析
- 学习路径、入门教程

### research-to-diagram
深度调研主题并自动生成知识关系图谱 PDF。

**触发词**：
- 研究...并做图
- 深度分析...并可视化
- 生成知识图谱
- 生成关系图
- 做成结构图 PDF

## 使用方法

将对应目录复制到 `~/.claude/skills/` 目录下即可使用。

```bash
# 示例：安装 deep-research
cp -r deep-research ~/.claude/skills/
```
